# Assignment Workflow Portal (Backend)

Backend API for Assignment Workflow Portal built with Node.js and Express.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Setup

1. Clone repo
2. Install dependencies

npm install

3. Create .env file

PORT=5000
MONGO_URI=mongodb://localhost:27017/assignmentPortal
JWT_SECRET=secret

4. Run server

npm run dev

Currently initial setup will update it
